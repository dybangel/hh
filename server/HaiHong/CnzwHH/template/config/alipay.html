参数配置 >> 提现参数配置
开启提现 {ke.function.isopen/} 提现次数 同用户每天最多允许提现次数,0为不限制
重复提现 {ke.function.iRepeat/} 提现总额 用户每日最多允许提现金额
提现金额 (单位元)单次提现最小金额 提现金额 (单位元)单次提现最大金额
提现时间 (小时分钟)每日开始提现的时间 结束时间 (小时分钟)每日结束提现的时间
参数配置 >> 支付宝提现设置
开启支付宝 {ke.function.isAlipay/} 关闭后将不再使用支付宝提现功能 订单审核 {ke.function.AliDisplay/} 提现方式 {ke.function.AlIsOnLine/}
*支付宝商家账号 审核金额 (元)提现超过当前金额需要审核
*支付宝商户公钥
