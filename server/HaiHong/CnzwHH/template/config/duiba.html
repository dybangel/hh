兑吧参数
参数配置 >> 兑吧参数 >> 基本配置
开启兑吧 {ke.function.isopen/} 关闭以后用户将无法使用兑吧兑换积分
兑换时间 - 每天允许用户兑换时间点
兑换金额 - 每天允许用户兑换时间点
兑换次数 单个用户每天允许兑换的次数,0为不限制
参数配置 >> 兑吧参数 >> 对接令牌
对接AppID 兑吧兑换接口使用appId
对接令牌 兑吧对接使用接口令牌appsecret
参数配置 >> 兑吧参数 >> 回调地址
兑吧接口说明
积分消费接口地址∶{ke.function.siteurl/}{app}/api/duiba/request.aspx
结果通知接口地址∶{ke.function.siteurl/}{app}/api/duiba/deal.aspx
违规查询接口地址∶{ke.function.siteurl/}{app}/api/duiba/ofline.aspx
以上三个接口地址再与兑吧对接是配置接口处填写，请勿随意更改！
